## 📋 What is FireRed-OCR?

FireRed-OCR is a simple application that helps you read text from images or screenshots. It can quickly recognize and convert text from pictures into editable text on your computer. ## 📝 How to Use FireRed-OCR

Using FireRed-OCR is simple and intuitive.

1. **Prepare Your Image**  
   Find an image or screenshot that contains text you want to read. You can use your phone, camera, or any image file saved on your computer.

2. **Load the Image**  
   In the FireRed-OCR window, click the **Open Image** button. Navigate to the folder where your image is saved, select it, and click **Open**.

3. **Start Text Recognition**  
   Click the **Start OCR** button. The software will scan the image and look for text on it.

4. **View the Extracted Text**  
   Once the scanning finishes, you will see the text appear inside the app. You can edit it, copy it, or save it as a text file.

5. **Save or Copy Text**  
   Use the **Copy** button to place the text on your clipboard. You can then paste it into other software like Word, Notepad, or email. Alternatively, use the **Save Text** option to store it on your computer as a `.txt` file.
